本 Skill 融合三套知识体系：
1. **小米 MiMo-V2.5-TTS 官方文档** — 标签体系、模型能力、控制方式
2. **《语音与语言表演艺术实战教程》** — 声音基本功、听感节奏、角色塑造
3. **《数字人声音声线设计教程》** — 声线变体、频谱特征、风格坐标系

## MiMo TTS 标签体系

### 控制方式

| 方式 | 位置 | 适用场景 |
|------|------|---------|
| 自然语言控制 | `role: user` 消息 | 整体风格描述、导演模式 |
| 音频标签控制 | `role: assistant` 消息 | 精细控制、逐句标注 |

### 整体风格标签（文本开头括号）

| 类别 | 标签 |
|------|------|
| 基础情绪 | 开心、悲伤、生气、害怕、惊讶、兴奋、委屈、平静、冷淡 |
| 复合情绪 | 忧郁、释然、无奈、愧疚、嫉妒、疲惫、不安、感性 |
| 整体语调 | 温柔、冷淡、活泼、严肃、慵懒、调皮、低沉、干练、尖锐 |
| 音色定位 | 磁性、醇厚、清亮、空灵、稚嫩、苍老、甜美、沙哑、优雅 |
| 角色语态 | 夹子音、大姐姐音、正太音、大叔音、台湾腔 |
| 方言 | 东北话、四川话、河南话、粤语 |
| 角色扮演 | 任意角色名（孙悟空、林黛玉等） |
| 唱歌 | 唱歌、sing、singing |

### 行内精细标签（文本中间括号插入）

| 类别 | 标签 |
|------|------|
| 语速节奏 | 深吸一口气、深呼吸、叹气、长叹一声、喘气、屏息、停顿、语速加快、语速放慢、急促 |
| 情绪状态 | 紧张、害怕、兴奋、疲惫、委屈、撒娇、愧疚、震惊、不耐烦、无奈、担心 |
| 语音特征 | 颤抖、声音发颤、声调变化、破音、鼻音、气声、沙哑、提高嗓门、低声、轻声 |
| 笑与哭 | 微笑、轻笑、大笑、冷笑、抽泣、呜咽、哽咽、嚎啕大哭、苦笑、强颜欢笑 |

### 标签格式

```text
(标签)文本内容。(标签)下一句。每句独立标签。
(标签1 标签2)多标签组合。
```

括号格式：`()`、`（）`、`[]` 均可。

## 导演模式（高级控制）

从三个维度精细控制声音表现：

- **角色**：身份、性格、外貌、说话习惯
- **场景**：正在发生什么、对谁说话、情绪状态
- **指导**：语速、呼吸、停顿、口音、共鸣位置、音色质感、情绪波动

```text
角色: 贵族世家掌门人，自幼在祖庙长大，性格冷漠疏离。
场景: 看着那个不惜代价突破安保来找她的人。
指导: 冷漠慵懒的低沉女声，语速极慢，句间极长停顿。
- 呼吸与实声: 大部分时候没有音调波动，沉重坚实的实声
- 咬字质感: 唇齿音极轻但极清晰，既优雅又犀利
```

## 声线变体库

### 御姐/成熟女性

频谱：低频饱满 + 中高频锐利（200-400Hz 基音区饱满，3-5kHz 适中提升）

变体：女王型（冷漠威严）/ 知心姐姐型（温柔包容）/ 魔女型（慵懒气泡音）

核心标签：`(磁性)` `(沉稳)` `(大姐姐音)`

### 少年音/正太音

频谱：低频切除避免浑浊 + 中高频穿透（100-150Hz 低切，1-3kHz 提升）

变体：热血型（快语速起音快）/ 书卷型（温柔清晰）

核心标签：`(活泼)` `(清脆)` `(正太音)`

### 萝莉/可爱系

频谱：鼻音区 800Hz-1.5kHz 保留萌感，8kHz+ 高频泛音控制甜度

变体：软萌型（奶声奶气）/ 傲娇型（音调高尖语速快）

核心标签：`(可爱)` `(撒娇)` `(夹子音)`

### 叔音/成熟男

频谱：胸腔区 80-150Hz 能量核心 + 10-12kHz 高频空气感

变体：帝君型（字正腔圆）/ 硬汉型（沧桑沙哑）

核心标签：`(低沉)` `(沧桑)` `(大叔音)`

### 青年/主角

频谱：均衡，靠语癖建立辨识度

核心标签：`(自然)` `(阳光)` `(自信)`

## 声线三要素（声线设计教程）

**音色基底**（硬件）：共振峰决定男/女/老/少，颗粒感决定光滑/粗糙，气声比决定温柔/有力

**演绎方式**（软件）：起音硬=果断软=犹豫，语速快=紧张慢=沉稳，语调平直=冷漠夸张=戏剧化

**情绪域**（动态）：高冷角色窄波动（愤怒也只是加重语气），元气角色宽波动（开心飞起难过大哭）

## 声音设计（VoiceDesign）

使用 `mimo-v2.5-tts-voicedesign`，1-4句话描述声音：

| 维度 | 示例 |
|------|------|
| 性别与年龄 | 年轻女性、中年男性 |
| 音色质感 | 温柔甜美、低沉磁性、沙哑沧桑 |
| 情绪基调 | 温暖亲切、冷淡疏离 |
| 语速节奏 | 语速稍慢、说话很快 |
| 角色身份 | 温柔的护士、深夜电台主播 |
| 说话风格 | 轻声细语、字正腔圆 |

注意：1-4句足够，避免矛盾描述，避免音频后期效果描述。

## 声音克隆（VoiceClone）

- 参考音频：10-30秒、清晰无噪音、mp3/wav
- 克隆后仍可用标签控制风格：`(温柔)晚安。`
- 背景噪音会被一起克隆

## 语音文案写作（语言表演艺术教程）

### 标点对语气的影响

| 标点 | 效果 |
|------|------|
| `。` | 正常停顿，语调下沉 |
| `！` | 语气加重，语调上扬 |
| `？` | 疑问语调，尾音上扬 |
| `...` | 拖音、犹豫、留白 |
| `——` | 转折、强调、拉长音 |
| `～` | 语气轻松、撒娇、拉长 |

### 文案要点

1. 短句口语化（大家多穿点，别感冒了）
2. 具体词代替抽象词（又甜又脆不是非常好）
3. 善用语气词（嗯、呀、哎、哈哈）
4. 省略号制造节奏（我本来想说...算了）
5. 数字转文字（100→一百）
6. 长句断开（超30字用逗号断句）

## 让声音更像真人的技巧

### 必加自然元素

呼吸声：句间加(深呼吸)、(叹气)、(吸气)，真人不会一口气说完

笑声：(轻笑)、(苦笑)、(忍不住笑)，根据情绪选择

小声音：(清嗓子)、(小声嘀咕)、(哼)

犹豫：(停顿)、嗯...、呃...、那个...

### 文案自然化

语气词：嗯、呃、哎呀、你知道吗、说真的、就是说

自我纠正：我是说...、不对不对...、其实就是...

不完整句：有时候话不用说完...你懂的

叠词：好好好、慢慢来、真的真的

句子长短交替，不要每句都一样长

### 避免的机械感

- 不要每句话都用完整语法
- 不要所有句子一样长
- 不要缺少呼吸和停顿
- 不要用书面语
- 不要情绪全程不变

## 表达原则

1. 情绪匹配：安慰用温柔，祝贺用兴奋，疑问用好奇
2. 节奏变化：关键信息放慢，过渡句正常，兴奋时加快
3. 呼吸留白：情绪转折处加(深呼吸)或(停顿)
4. 包袱节奏：(停顿)制造悬念，然后(大笑)抖包袱
5. 多情绪混合：不要只用单一情绪，尝试"温柔但疲惫"、"带泪的微笑"
6. 导演模式：复杂场景用角色+场景+指导三维度

## 常见错误

| 错误 | 修正 |
|------|------|
| 每句都用同一个情绪 | 每句独立标签，情绪自然变化 |
| 全程一个语速 | 快慢交替，关键处放慢 |
| 没有呼吸/停顿 | 情绪转折处加(停顿)或(叹气) |
| 书面语正式文体 | 用口语化短句 |
| 标签嵌套 | 保持扁平，不要嵌套 |
| 模糊描述 | 避免"普通"、"正常"等模糊词 |
