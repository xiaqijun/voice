# TTS 风格控制 Skill

融合小米 MiMo-V2.5-TTS 官方文档、《语音与语言表演艺术实战教程》、《数字人声音声线设计教程》三套知识体系。

## 一、MiMo TTS 官方标签体系

### 控制方式
1. **自然语言控制** → 放在 `role: user` 消息中（风格描述）
2. **音频标签控制** → 放在 `role: assistant` 消息中（文本内嵌标签）

### 整体风格标签（文本开头括号）

| 类别 | 可用标签 |
|------|---------|
| 基础情绪 | 开心, 悲伤, 生气, 害怕, 惊讶, 兴奋, 委屈, 平静, 冷淡 |
| 复合情绪 | 忧郁, 释然, 无奈, 愧疚, 嫉妒, 疲惫, 不安, 感性 |
| 整体语调 | 温柔, 冷淡, 活泼, 严肃, 慵懒, 调皮, 低沉, 干练, 尖锐 |
| 音色定位 | 磁性, 醇厚, 清亮, 空灵, 稚嫩, 苍老, 甜美, 沙哑, 优雅 |
| 角色语态 | 夹子音, 大姐姐音, 正太音, 大叔音, 台湾腔 |
| 方言 | 东北话, 四川话, 河南话, 粤语 |
| 角色扮演 | 孙悟空, 林黛玉（任意角色名） |
| 唱歌 | 唱歌, sing, singing |

| 类别 | 可用标签 |

### 官方示例风格
```
(叹气)这么多年了，当我再次走上那条街，心里突然空了一块。
(慵懒)让我再睡五分钟...就五分钟，真的，最后一次。
(磁性)夜已经深了，但城市还在呼吸。今晚陪你的是我。欢迎收听《午夜电台》。
(东北话)哎呀妈呀，今天也太冷了吧！你知道那风，嗖嗖的跟刀子似的，往脸上割！
(粤语)这个真的太好吃了！你一试就忘不了！
(唱歌)原谅我这一生不羁放纵爱自由...
```

### 行内精细控制示例
```
(紧张，深吸一口气)呼...冷静，冷静。只是个面试...(语速加快，嘀咕)自我介绍我排练了五十遍，应该没问题。
(疲惫至极，有气无力)主人...到了叫我...(深深叹气)我先眯一会儿。这加班加得魂都要散了。
如果我...(停顿片刻)哪怕再多坚持一秒钟，结局会不会不一样？(苦笑)呵，没有如果了。
(因寒冷而急促呼吸)呼——呼——这，这大兴安岭的雪...(咳嗽)真能冻到骨头里...
(提高嗓门喊)大姐！这鱼新鲜的！今早刚打上来的！哎！你别翻别翻！翻烂了你得赔！
```

## 二、声线三要素（声线设计教程）

### 音色基底 (Timbre) —— 硬件
| 参数 | 含义 | 标签映射 |
|------|------|---------|
| 共振峰 | 低=成熟/体型大，高=年轻/体型小 | 叔音 vs 萝莉 |
| 颗粒感 | 光滑 vs 粗糙 | 磁性/沙哑 vs 清透/甜美 |
| 气声比 | 气多=性感温柔，声多=有力自信 | 低语/气声 vs 坚定 |

### 演绎方式 (Delivery) —— 软件
| 参数 | 两极 | 标签映射 |
|------|------|---------|
| 起音 | 硬=冲/果断，软=犹豫/温柔 | 急促 vs 慵懒 |
| 语速 | 快=紧张，慢=沉稳 | 语速加快 vs 语速放慢 |
| 语调曲线 | 平直=冷漠，夸张=戏剧化 | 冷淡 vs 活泼 |

### 情绪域 (Emotion Range) —— 动态
- 窄波动：高冷角色，即使愤怒也只是语气加重
- 宽波动：元气角色，开心飞起，难过大哭

## 三、声线变体库

### 御姐/成熟女性
- 频谱：低频饱满 + 中高频锐利
- 变体：女王型(冷漠威严) / 知心姐姐型(温柔包容) / 魔女型(慵懒气泡音)
- 标签：`(磁性)` `(沉稳)` `(温柔)` `(慵懒)` `(大姐姐音)`

### 少年音/正太音
- 频谱：低频切除 + 中高频极强(穿透力)
- 变体：热血型(大嗓门快语速) / 书卷型(温柔清晰)
- 标签：`(活泼)` `(清脆)` `(热血)` `(正太音)`

### 萝莉/可爱系
- 频谱：鼻音区(800Hz-1.5kHz)保留萌感
- 变体：软萌型(奶声奶气) / 傲娇型(音调高尖)
- 标签：`(可爱)` `(撒娇)` `(稚嫩)` `(夹子音)`

### 叔音/成熟男
- 频谱：胸腔区(80-150Hz)能量核心 + 高频空气感
- 变体：帝君型(字正腔圆) / 硬汉型(沧桑沙哑)
- 标签：`(低沉)` `(磁性)` `(沧桑)` `(大叔音)`

### 青年/主角
- 均衡设计，靠语癖建立辨识度
- 标签：`(自然)` `(阳光)` `(自信)`

## 四、MiMo TTS 多层控制能力

### 多风格切换
同一段语音中自然过渡：播报 → 耳语 → 呐喊

### 多情绪混合
支持复杂情绪："压抑的愤怒"、"带泪的微笑"、"温柔但疲惫"、"狂躁中的温柔"

### 多粒度控制
- 段落级：整体语调
- 句子级：节奏
- 词语级：重音
- 字符级：哽咽、拖音、某个字的气声

### 导演模式（高级）
三个维度精细控制：
- **角色**：身份、性格、外貌、说话习惯
- **场景**：正在发生什么、对谁说、情绪状态
- **指导**：语速、呼吸、停顿、口音、共鸣位置、音色质感、情绪波动

## 五、标签使用规范

### 整体风格（开头）
```
(开心)你好呀！
(温柔 慢速)晚安，好好休息。
(御姐 磁性)你终于来了。
(东北话)哎呀妈呀！
```

### 行内精细（中间）
```
(开心)太好了！(停顿)我就知道你可以的！
(叹气)唉...(无奈)没办法了。
(紧张 深呼吸)我...我有点害怕。
(微笑)你真可爱。
```

### 复合标签
```
(温柔 低语 慢速)晚安...
(热血 活泼 快速)冲冲冲！
(冷淡 磁性 沉稳)无所谓。
(委屈 撒娇 慢速)你怎么才来呀...
```

## 七、语音表达原则

1. **情绪匹配**：根据内容选情绪，安慰用温柔，祝贺用兴奋
2. **节奏变化**：关键信息放慢，过渡句正常，兴奋时加快
3. **呼吸留白**：情绪转折处加(深呼吸)或(停顿)
4. **包袱节奏**：讲笑话时(停顿)制造悬念，然后(大笑)抖包袱
5. **高级感**：关注低频质感和高频空气感
6. **多情绪混合**：不要只用单一情绪，尝试"温柔但疲惫"、"带泪的微笑"
7. **导演模式**：复杂场景用角色+场景+指导三维度描述
