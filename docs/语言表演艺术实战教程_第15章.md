# 第15章｜写作与台本：把“能说”变成“可复制”

> **核心格言**：所有的即兴，都是蓄谋已久的准备。高手和新手的区别在于：新手的稿子是写给眼睛看的文章，高手的稿子是写给嘴巴和耳朵的“乐谱”。

## 1. 开篇：为什么必须要练“写”？

在前面的章节中，我们训练了声音、呼吸、节奏和幽默感。到了这一章，我们要解决**“稳定性”**和**“可复制性”**的问题。

很多初学者认为口才好等于“出口成章、由于不决”，这是一种误解。
- **电台主播**看似随意的闲聊，面前通常摆着精确到秒的流程单（Rundown）。
- **脱口秀演员**看似现挂的爆梗，背后可能经历了上百次的改稿和打磨。
- **相声演员**的“贯口”，一个字都不能错，那是千锤百炼的剧本。

**本章的学习目标**：
1.  **思维转换**：从“书面语思维”彻底转变为“口语化思维”（Writing for the Ear）。
2.  **台本工程**：掌握逐字稿、提纲稿、提示卡（Cue Card）的制作规范。
3.  **节奏可视化**：学会使用符号系统，把呼吸、重音、停顿“写”进稿子里。
4.  **场景化写作**：针对不同场合（主持、播客、讲故事）的结构模板。

## 2. 核心技法论述

### 2.1 书面语 vs. 口语体：重新编程你的大脑

写作是给人看的，可以回读，结构可以复杂（嵌套从句）。口语是给人听的，稍纵即逝，必须线性向前。

**核心法则 (Rules of Thumb)**：

1.  **拒绝“俄罗斯套娃”句式**：
    *   *书面*：“那个穿着红色外套的、手里拿着冰激凌的、正在哭泣的小男孩是我的外甥。”（定语过长，主语滞后）
    *   *口语*：“看到那个穿红外套的小男孩了吗？手里拿着冰激凌哭的那个，哎对，那是我外甥。”（拆解信息，逐步引导画面）

2.  **善用“冗余”和“路标”**：
    *   听觉记忆很短。在书面语中，重复是啰嗦；在口语中，重复是**强调**。
    *   使用“路标词”：*“第一点...”、“接下来...”、“举个例子...”、“总而言之...”*，告诉听众现在在哪儿。

3.  **把“虚词”换成“实词”**：
    *   不要说“他在某种程度上表现出了愤怒”，要说“他气得直拍桌子”。
    *   动词和名词比形容词更有力量。

### 2.2 台本的格式工程学

台本不仅仅是文字内容，**排版**直接影响你读稿时的状态。

*   **字号**：必须大！建议 14pt - 16pt。不要让自己眯着眼睛找字，那会让你皱眉，声音变紧。
*   **行距**：1.5倍或2倍行距。留出空间做笔记（画停顿、写情绪）。
*   **页码**：每一页都要标页码（如 Page 1 of 5）。防止掉地上一片混乱。
*   **换页不换句**：**严禁**把一句话切断在两页纸上。翻页声会打断情绪，翻页时的视线移动会导致喷麦或音量变化。

### 2.3 这里的乐谱：台本标记系统 (Script Notation)

这是本章最硬核的技能。把你的台本变成一张乐谱。

| 符号 | 含义 | 作用 | 示例 |
| :--- | :--- | :--- | :--- |
| **/** | 短停顿 | 逻辑断句，不换气 | 大家看 **/ 前面这辆车** |
| **//** | 长停顿 | 强调，甚至换气 | 这件事 **/ 简直不可思议** |
| **V** | 气口 | 必须吸气的地方 | (V) 各位听众晚上好 |
| **<u>下划线</u>** | 重读 | 强调关键词 | 这不是<u>我想</u>要的结果 |
| **...** | 拖音 | 语速放慢，意味深长 | 他慢慢地... 转过头来 |
| **[ ]** | 动作/情绪 | 提示非语言行为 | [轻笑] 谁说不是呢？ |
| **~** | 波浪线 | 语调上扬或柔和 | 宝贝~ 睡觉啦 |
| **>>>** | 加速 | 紧张感或贯口 | >>>(车如流水马如龙) |

### 2.4 三种典型台本结构

#### A. 播客/电台/有声书（全逐字稿）
要求精确，每一句话都设计好。
> **结构**：Hook（钩子，前30秒抓住人） -> Intro（我是谁，聊什么） -> Body（分点叙述，故事+评论） -> Outro（总结+召唤行动）。

#### B. 活动主持/直播（手卡关键词稿）
全写下来会照本宣科，眼神无法和观众交流。
> **写法**：只写**首尾句**和**流程关键词**。
> *   *开头*：“尊敬的各位来宾...” (防止紧张忘词，写全)
> *   *中间*：1. 抽奖 (规则：摇一摇) -> 2. 张总发言 (Title：CTO) -> 3. 游戏环节
> *   *结尾*：“今天的活动到此结束...” (写全)

#### C. 相声/脱口秀（结构稿）
> **结构**：
> *   **垫话**：闲聊，拉近关系，引出话题。
> *   **瓢把儿**：引入正活的过渡段。
> *   **正活**：核心段子，三翻四抖。
> *   **底**：最高潮的包袱，笑声中鞠躬下台。

### 2.5 幽默写作公式：铺垫与反转

不要试图“写”好笑，要“设计”好笑。

1.  **Setup (铺垫)**：建立一个逻辑A（也就是听众脑补的正常走向）。
2.  **Connector (连接点)**：一个双关词、歧义句或模糊的情境。
3.  **Shatter (打碎/反转)**：揭示逻辑B（出乎意料但合理的解释）。

> *示例*：
> *铺垫*：我爷爷甚至能记得他10岁那年发誓要报效祖国的那一天。
> *反转*：但他记不住我是谁。

## 3. 本章小结

1.  **台本是听觉服务**：如果你的稿子默读很顺，但朗读很累，那就是**烂稿子**。必须改短句、加气口。
2.  **格式即战斗力**：大字号、宽行距、换页不切句，这能救你的命。
3.  **标记是灵魂**：不带标记的台本只是TXT文档，带了重音、停顿和情绪标记的才是脚本。
4.  **结构先行**：先用大纲搭骨架（起承转合），再用口语填血肉，最后用修辞做装饰。

## 5. 常见陷阱与错误 (Gotchas)

### 5.1 “默读欺骗” (The Silent Reading Trap)
*   **陷阱**：你在心里默读稿子，觉得行云流水，文采飞扬。
*   **真相**：一旦张嘴朗读，发现舌头打结，气口不够。因为默读是跳跃式的，朗读是物理运动。
*   **调试技巧**：**Write out loud.** 写稿的时候就必须念出声。如果你的嘴巴在打架，你的稿子就有问题。

### 5.2 “时间盲区” (Timing Blindness)
*   **陷阱**：台本上写了“简单聊两句”，结果嘉宾聊了10分钟，导致后面流程腰斩。
*   **调试技巧**：永远不要相信“简单聊聊”。在台本上按字数估算时间，并标注`[Cut点]`（如果超时，从哪里切断）。

### 5.3 “括号狂魔” (Over-direction)
*   **陷阱**：给每一句话都加 `[悲伤地]`、`[愤怒地]`、`[若有所思地]`。
*   **真相**：这会限制你的表演，让你变得像个机器人。
*   **调试技巧**：好的台本，情绪是蕴含在**选词**和**标点**里的。如果你写“滚！”，不需要标注`[愤怒地]`；如果你写“滚...”，那可能是`[失望地]`。让文字本身说话。

### 5.4 “生僻字地雷”
*   **陷阱**：为了显有文化，用了“耄耋”、“饕餮”、“以飨读者”等词。
*   **真相**：在紧张的高压环境下，你的大脑会短路，看到生僻字会卡壳，甚至读错音（把“在这个时刻”读成“在介个时刻”）。
*   **调试技巧**：**K.I.S.S. 原则** (Keep It Simple, Stupid)。除非是成语接龙，否则尽量用初中生都能听懂的词汇。

### 5.5 “我是谁？” (Identity Crisis)
*   **陷阱**：写主持稿像播音员，写播客稿像相声演员，写相声稿像作报告。
*   **调试技巧**：写稿前先写下 **Persona（人设）**。
    *   *我是知心大姐* -> 用词温软、多问句。
    *   *我是新闻主播* -> 用词准确、短促、肯定句。
    *   *我是相声捧哏* -> 话少、精辟、不抢戏。
