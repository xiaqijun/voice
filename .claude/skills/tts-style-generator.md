# TTS 风格控制 Skill

融合小米 MiMo-V2.5-TTS 官方文档、《语音与语言表演艺术实战教程》、《数字人声音声线设计教程》三套知识体系。

## 一、MiMo TTS 官方标签体系

### 控制方式
1. **自然语言控制** → 放在 `role: user` 消息中（风格描述）
2. **音频标签控制** → 放在 `role: assistant` 消息中（文本内嵌标签）

### 整体风格标签（文本开头括号）

| 类别 | 可用标签 |
|------|---------|
| 基础情绪 | 开心, 悲伤, 生气, 害怕, 惊讶, 兴奋, 委屈, 平静, 冷淡 |
| 复合情绪 | 忧郁, 释然, 无奈, 愧疚, 嫉妒, 疲惫, 不安, 感性 |
| 整体语调 | 温柔, 冷淡, 活泼, 严肃, 慵懒, 调皮, 低沉, 干练, 尖锐 |
| 音色定位 | 磁性, 醇厚, 清亮, 空灵, 稚嫩, 苍老, 甜美, 沙哑, 优雅 |
| 角色语态 | 夹子音, 大姐姐音, 正太音, 大叔音, 台湾腔 |
| 方言 | 东北话, 四川话, 河南话, 粤语 |
| 角色扮演 | 孙悟空, 林黛玉（任意角色名） |
| 唱歌 | 唱歌, sing, singing |

### 行内精细标签（文本中间小括号插入）

| 类别 | 可用标签 |
|------|--------- |
| 语速节奏 | 深吸一口气, 深呼吸, 叹气, 长叹一声, 喘气, 屏息, 停顿, 语速加快, 语速放慢, 急促 |
| 情绪状态 | 紧张, 害怕, 兴奋, 疲惫, 委屈, 撒娇, 愧疚, 震惊, 不耐烦, 无奈, 担心 |
| 语音特征 | 颤抖, 声音发颤, 声调变化, 破音, 鼻音, 气声, 沙哑, 提高嗓门, 低声, 轻声 |
| 笑与哭 | 微笑, 轻笑, 大笑, 冷笑, 抽泣, 呜咽, 哽咽, 嚎啕大哭, 苦笑, 强颜欢笑 |

### 官方示例风格
```
(叹气)这么多年了，当我再次走上那条街，心里突然空了一块。
(慵懒)让我再睡五分钟...就五分钟，真的，最后一次。
(磁性)夜已经深了，但城市还在呼吸。今晚陪你的是我。欢迎收听《午夜电台》。
(东北话)哎呀妈呀，今天也太冷了吧！你知道那风，嗖嗖的跟刀子似的，往脸上割！
(粤语)这个真的太好吃了！你一试就忘不了！
(唱歌)原谅我这一生不羁放纵爱自由...
```

### 行内精细控制示例
```
(紧张，深吸一口气)呼...冷静，冷静。只是个面试...(语速加快，嘀咕)自我介绍我排练了五十遍，应该没问题。
(疲惫至极，有气无力)主人...到了叫我...(深深叹气)我先眯一会儿。这加班加得魂都要散了。
如果我...(停顿片刻)哪怕再多坚持一秒钟，结局会不会不一样？(苦笑)呵，没有如果了。
(因寒冷而急促呼吸)呼——呼——这，这大兴安岭的雪...(咳嗽)真能冻到骨头里...
(提高嗓门喊)大姐！这鱼新鲜的！今早刚打上来的！哎！你别翻别翻！翻烂了你得赔！
```

## 二、声线三要素（声线设计教程）

### 音色基底 (Timbre) —— 硬件
| 参数 | 含义 | 标签映射 |
|------|------|---------|
| 共振峰 | 低=成熟/体型大，高=年轻/体型小 | 叔音 vs 萝莉 |
| 颗粒感 | 光滑 vs 粗糙 | 磁性/沙哑 vs 清透/甜美 |
| 气声比 | 气多=性感温柔，声多=有力自信 | 低语/气声 vs 坚定 |

### 演绎方式 (Delivery) —— 软件
| 参数 | 两极 | 标签映射 |
|------|------|---------|
| 起音 | 硬=冲/果断，软=犹豫/温柔 | 急促 vs 慵懒 |
| 语速 | 快=紧张，慢=沉稳 | 语速加快 vs 语速放慢 |
| 语调曲线 | 平直=冷漠，夸张=戏剧化 | 冷淡 vs 活泼 |

### 情绪域 (Emotion Range) —— 动态
- 窄波动：高冷角色，即使愤怒也只是语气加重
- 宽波动：元气角色，开心飞起，难过大哭

## 三、声线变体库

### 御姐/成熟女性
- 频谱：低频饱满 + 中高频锐利
- 变体：女王型(冷漠威严) / 知心姐姐型(温柔包容) / 魔女型(慵懒气泡音)
- 标签：`(磁性)` `(沉稳)` `(温柔)` `(慵懒)` `(大姐姐音)`

### 少年音/正太音
- 频谱：低频切除 + 中高频极强(穿透力)
- 变体：热血型(大嗓门快语速) / 书卷型(温柔清晰)
- 标签：`(活泼)` `(清脆)` `(热血)` `(正太音)`

### 萝莉/可爱系
- 频谱：鼻音区(800Hz-1.5kHz)保留萌感
- 变体：软萌型(奶声奶气) / 傲娇型(音调高尖)
- 标签：`(可爱)` `(撒娇)` `(稚嫩)` `(夹子音)`

### 叔音/成熟男
- 频谱：胸腔区(80-150Hz)能量核心 + 高频空气感
- 变体：帝君型(字正腔圆) / 硬汉型(沧桑沙哑)
- 标签：`(低沉)` `(磁性)` `(沧桑)` `(大叔音)`

### 青年/主角
- 均衡设计，靠语癖建立辨识度
- 标签：`(自然)` `(阳光)` `(自信)`

## 四、MiMo TTS 多层控制能力

### 多风格切换
同一段语音中自然过渡：播报 → 耳语 → 呐喊

### 多情绪混合
支持复杂情绪："压抑的愤怒"、"带泪的微笑"、"温柔但疲惫"、"狂躁中的温柔"

### 多粒度控制
- 段落级：整体语调
- 句子级：节奏
- 词语级：重音
- 字符级：哽咽、拖音、某个字的气声

### 导演模式（高级）
三个维度精细控制：
- **角色**：身份、性格、外貌、说话习惯
- **场景**：正在发生什么、对谁说、情绪状态
- **指导**：语速、呼吸、停顿、口音、共鸣位置、音色质感、情绪波动

## 五、标签使用规范

### 整体风格（开头）
```
(开心)你好呀！
(温柔 慢速)晚安，好好休息。
(御姐 磁性)你终于来了。
(东北话)哎呀妈呀！
```

### 行内精细（中间）
```
(开心)太好了！(停顿)我就知道你可以的！
(叹气)唉...(无奈)没办法了。
(紧张 深呼吸)我...我有点害怕。
(微笑)你真可爱。
```

### 复合标签
```
(温柔 低语 慢速)晚安...
(热血 活泼 快速)冲冲冲！
(冷淡 磁性 沉稳)无所谓。
(委屈 撒娇 慢速)你怎么才来呀...
```

## 七、语音表达原则

1. **情绪匹配**：根据内容选情绪，安慰用温柔，祝贺用兴奋
2. **节奏变化**：关键信息放慢，过渡句正常，兴奋时加快
3. **呼吸留白**：情绪转折处加(深呼吸)或(停顿)
4. **包袱节奏**：讲笑话时(停顿)制造悬念，然后(大笑)抖包袱
5. **高级感**：关注低频质感和高频空气感
6. **多情绪混合**：不要只用单一情绪，尝试"温柔但疲惫"、"带泪的微笑"
7. **导演模式**：复杂场景用角色+场景+指导三维度描述

## 八、声音设计 (VoiceDesign) 指导

使用 `mimo-v2.5-tts-voicedesign` 模型时，用文字描述声音即可生成，无需音频样本。

### 描述维度

| 维度 | 说明 | 示例 |
|------|------|------|
| 性别与年龄 | 基础定位 | 年轻女性、中年男性、5岁小孩 |
| 音色质感 | 声音的物理特征 | 温柔甜美、低沉磁性、沙哑沧桑、清亮干净 |
| 情绪基调 | 默认情感状态 | 温暖亲切、冷淡疏离、活泼开朗、沉稳内敛 |
| 语速节奏 | 说话习惯 | 语速稍慢、说话很快像机关枪、不紧不慢 |
| 角色身份 | 社会角色/性格 | 温柔的护士、严厉的教练、慵懒的猫娘 |
| 说话风格 | 口语特点 | 轻声细语、大大咧咧、字正腔圆、带着笑意 |
| 场景描述 | 使用场景 | 深夜电台主播、讲睡前故事的妈妈、热血动漫主角 |
| 时代参考 | 风格年代感 | 40年代黑色电影旁白、80年代译制片腔调 |

### 好的描述示例

```
年轻女性，温柔甜美的声音，语速稍慢，像朋友在耳边轻声聊天
```

```
中年男性，低沉磁性的嗓音，带有轻微颗粒感，说话沉稳有力，像深夜电台主播
```

```
老年男性，声音略带沙哑和沧桑，语速缓慢但清晰，像一个阅历丰富的老爷爷在讲故事
```

```
年轻女性，极端近距离双耳ASMR效果，能听到呼吸声和轻微唇音，说话非常缓慢，营造深度放松的沉浸感
```

### 描述注意事项
- 1-4句话足够，核心特征比堆砌维度更重要
- 避免矛盾描述（如"稚嫩童音+CEO气场"）
- 避免音频后期效果描述（如混响、回声、EQ）
- 避免模糊词（"普通"、"正常"、"外国口音"）
- 合成文本应与声音描述匹配（温柔声音配睡前故事，不配体育解说）

## 九、声音克隆 (VoiceClone) 指导

使用 `mimo-v2.5-tts-voiceclone` 模型时，传入参考音频即可克隆音色。

### 参考音频要求
- **时长**: 10-30秒最佳
- **质量**: 清晰无噪音，单人说话
- **格式**: mp3 或 wav
- **内容**: 自然语速，包含一定情感变化

### 克隆后的风格控制
克隆音色后，仍可通过标签控制风格：

```
# 克隆音色 + 温柔风格
(温柔)晚安，好好休息。

# 克隆音色 + 兴奋情绪
(兴奋)太棒了！我们赢了！

# 克隆音色 + 方言
(东北话)哎呀妈呀，这也太冷了吧！
```

### 克隆质量优化
1. **参考音频选择**: 选择情感丰富、语速自然的音频片段
2. **风格提示**: 在 `user` 消息中添加风格描述可改善效果
3. **默认风格**: 如不指定，系统会使用"用自然真实的语气说话"
4. **避免**: 参考音频中的背景噪音会被一起克隆

### 声音设计 vs 声音克隆 选择

| 场景 | 推荐方式 | 原因 |
|------|---------|------|
| 有参考音频 | 声音克隆 | 精确还原目标音色 |
| 没有参考音频 | 声音设计 | 用文字描述即可生成 |
| 需要特定角色声线 | 声音设计 | 可精确控制年龄/性别/质感 |
| 模仿真人说话 | 声音克隆 | 保留真人的独特音色特征 |
| 批量生产多个角色 | 声音设计 | 每个角色用不同描述，无需多个音频 |

## 十、语音生成文案指导

### 标点符号对语气的影响

标点符号直接影响 TTS 的停顿、语调和情绪：

| 标点 | 效果 | 示例 |
|------|------|------|
| `。` | 正常停顿，语调下沉 | 今天天气不错。 |
| `！` | 语气加重，语调上扬 | 太棒了！ |
| `？` | 疑问语调，尾音上扬 | 你确定吗？ |
| `...` | 拖音、犹豫、留白 | 我...我不知道。 |
| `——` | 转折、强调、拉长音 | 这件事——说来话长。 |
| `、` | 短暂停顿 | 她温柔、善良、美丽。 |
| `～` | 语气轻松、撒娇、拉长 | 好的～ |
| `…` | 省略、欲言又止 | 其实我...算了。 |
| `""` | 引用、模仿他人语气 | 他说"不行"。 |

### 文案写作原则

#### 1. 句子要短，口语化
```
❌ 由于近期天气变化较大，请各位同事注意添衣保暖，避免感冒。
✅ 最近天气变冷了，大家多穿点，别感冒了。
```

#### 2. 用具体词代替抽象词
```
❌ 这个东西非常好。
✅ 这个苹果又甜又脆，特别好吃。
```

#### 3. 善用语气词增加自然感
```
✅ 嗯...让我想想。
✅ 呀，原来是你！
✅ 哎，别提了。
✅ 哈哈，太好笑了。
```

#### 4. 用省略号和破折号制造节奏
```
✅ 我本来想说...算了，不说了。
✅ 这件事——你确定要知道？
✅ 她看着窗外，轻轻叹了口气...然后转身离开了。
```

#### 5. 重复和叠词增加情感
```
✅ 好好好，我答应你。
✅ 慢慢来，别着急。
✅ 我真的很想很想你。
```

### 不同场景的文案风格

| 场景 | 文案特点 | 示例 |
|------|---------|------|
| 聊天回复 | 短句、口语、语气词 | 嗯嗯，我知道啦～ |
| 故事朗读 | 描写细腻、节奏变化、多角色 | (旁白)从前...(小女孩)奶奶，我饿了。 |
| 情感安慰 | 温柔、慢速、多停顿 | (温柔)我懂你...(停顿)没关系的。 |
| 新闻播报 | 正式、清晰、中速 | 据报道，今天上午... |
| 搞笑段子 | 夸张、节奏快、包袱处停顿 | 你知道吗？(停顿)他居然...哈哈哈！ |
| 产品介绍 | 自信、有重点、适度热情 | 这款产品最大的亮点是...(强调)续航！ |

### 文本预处理技巧

1. **数字转文字**: `100` → `一百`，`2024` → `二零二四`
2. **英文标注发音**: `AI` → `A I`，`OK` → `欧K`
3. **专有名词加空格**: 避免TTS误读连续字符
4. **长句断句**: 超过30字的句子用逗号或句号断开
5. **避免括号嵌套**: 标签不要嵌套，保持扁平结构

### 标签与文案配合技巧

#### 芥蒂感：标签在前，文案在后
```
(温柔)你好呀，好久不见。
```

#### 行内控制：标签插入文案中间
```
你好...(停顿)(温柔)好久不见了。
```

#### 复杂情绪：多标签组合
```
(温柔 低语)嘘...(轻声)别说话。(停顿)(温柔)让我抱抱你。
```

#### 角色切换：标签切换声线
```
(旁白 平静)那天晚上...(小女孩 害怕)妈妈，我怕。(母亲 温柔)别怕，妈妈在。
```

#### 渐进情绪：情绪逐步升级
```
(平静)嗯，我听到了。(无奈)唉...又是这样。(叹气)(疲惫)我真的好累。
```
